  • 分别定义 base路由和业务路由. 业务路由里又可以定义版本路由，认证路由，不需要认证的路由，这样的话，在以后的CURD过程中，就可以把需要版本控制，权限控制等路由添加到自定的方法中去。

    // 1.顶一个InitSysRouter 初始化业务路由，基本路由，需要check得路由等等
func InitSysRouter( r *gin.Engine){
  g:=r.Group("")
  // init base router
  InitBaseRouter(g)
  // init 业务router(example)
  V1(g)
  // 无需认证
  
}
  
// 2. 定义基本路由
func InitBaseRouter(g *gin.RouterGroup) {
  g.GET("/",StaticHandler.HelloWorld)
  g.GET("/info",StaticHandler.InfoHandlerFunc)
}
  
  
//3. 定义业务版本管理,
